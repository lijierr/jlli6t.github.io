## 1.Copy local files to remote server
{% highlight bash %}
$scp local_file remore_username@remote_ip:remote_folder
{% endhighlight %}
<!--more-->

## 2. Copy file from remote server to local
{% highlight bash %}
$scp remote_username@remote_ip:remote_folder local_folder
{% endhighlight %}
